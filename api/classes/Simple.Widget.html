Simple.Widget Class

Extends Simple.Attribute
Defined in: widget.js:5
Module: widget

继承自Attribute类，为所有的SimpleUI的UI组件提供基类支持，拥有init,render,destroy生命周期. 这个基类提供了很多可以复写的方法，可以方便的实现自定义行为，从而降低工作量.

该类被设计为用于继承的基类，通常不会被直接实例化
